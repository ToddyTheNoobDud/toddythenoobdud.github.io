# Kenium Music Bot

Kennium is a music bot designed to enhance your music listening experience on various platforms. This README provides an overview of the project, its features, and instructions for setup and usage.

## Features

- **Changelog**: View the updates and changes made to the bot.
- **Destroy**: Remove the bot from the server.
- **Export**: Export your data from the bot.
- **Grab**: Capture the current track information.
- **Now Playing**: Display the currently playing track.
- **Play**: Play a specific track from the queue.
- **Import**: Import data into the bot.
- **Invite**: Invite the bot to your server.
- **Jump**: Jump to a specific track in the queue.
- **Lyrics**: Show the lyrics of the currently playing song.
- **Loop**: Loop the current track.
- **Pause**: Pause the currently playing track.
- **Ping**: Check the bot's latency.
- **Play File**: Play a file from your local system.
- **Queue**: View the current music queue.
- **Restart**: Restart the bot.
- **Resume**: Resume playback after pausing.
- **Search**: Search for tracks.
- **Seek**: Seek to a specific time in the current track.
- **Skip**: Skip to the next track in the queue.
- **Status**: Show the current status of the bot.
- **Stop**: Stop playback.
- **Volume**: Adjust the playback volume.

## Setup Instructions

1. **Clone the Repository**: 
   ```
   git clone <repository-url>
   ```

2. **Navigate to the Project Directory**: 
   ```
   cd kenium-music-bot
   ```

3. **Open `index.html`**: Open the `index.html` file to acess the interface

4. **Customize Styles**: Modify the `css/styles.css` file to change the apparance

5. **Add JavaScript Functionality**: Use the `js/scripts.js` file to add commands / remove

## Usage Guidelines

- Use the commands listed above to interact with the bot.
- Refer to the individual command HTML files in the `commands` directory for detailed usage instructions for each command.
